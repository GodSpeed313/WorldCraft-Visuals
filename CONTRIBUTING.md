# Contributing to WorldCraft-Visuals 🌌

First off, thank you for being here! Whether you're a pro or just "picking the brain" of the tech world like I am, I appreciate your interest in this project.

## How You Can Help 🛠️

### 1. Suggesting New Legends
Have a historical figure or live-action actor that would make a dope fusion? 
- Head over to our [Issues tab](https://github.com/GodSpeed313/WorldCraft-Visuals/issues).
- Look for the "Community Brainstorm" thread.
- Drop your suggestion and why you think they'd fit!

### 2. Code Contributions
If you want to add a feature or fix a bug:
- Fork the repository.
- Create a new branch for your feature.
- Submit a Pull Request (PR) and I'll review it as soon as I can.

### 3. Feedback & Learning
Since I'm an "Eternal Student," I welcome any advice on how to make this code cleaner or more efficient. Don't be shy!

## Our Vibe 💡
Let’s keep it positive, calm, and a little bit silly. This is a space for creativity and learning.

**Happy Building!**
